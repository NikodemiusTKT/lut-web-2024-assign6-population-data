# Implement HTML Structure

Create the basic HTML structure in `index.html` for displaying the population growth chart and input field.

**Acceptance Criteria:**
- The HTML file contains a `<div>` for the chart.
- An input field for municipality codes and a submit button are present.

**Steps to Complete:**
1. Add a `<div id="chart"></div>` for the chart.
2. Add an input field with `id="input-area"`.
3. Add a button with `id="submit-data"`.
